App Description
'트랜슬레이터 X'는 이미지, 음성, 문자 번역을 지원하는 강력한 번역 서비스입니다. 모국어로 자연스럽게 전달하려는 문구를 입력하거나, 말하거나, 사진을 찍어 올리면 앱이 자동으로 내용을 인식하고 해석해 번역 문장을 크게 발음해줍니다.

'트랜슬레이터 X'은 세계 어디를 여행하더라도 기본적인 의사소통이 가능하게 도와줍니다. 외국 여행이나 출장 시 공항이나 호텔, 수퍼마켓, 택시 등을 이용하는 분들께 좋습니다. 간단한 외국어 문구를 배우고 발음을 들으며 공부하려는 어린이나 성인 학습자에게도 유용합니다.

사물 모드 작동 절차:
- 사진을 촬영하세요
- 사진 내에서 사물의 이름을 확인하세요
- 선택한 언어로 번역된 결과를 받아보세요

이미지 모드 작동 방법:
- 어떤 것이든 인쇄된 텍스트를 촬영하세요
- 텍스트가 인식된 후 선택한 언어로 번역되었는지 확인하세요

보이스 모드 작동 방법:
- 마이크를 통해 모국어 문장을 말해주세요
- 선택한 언어로 번역된 결과를 받아보세요

텍스트 모드 작동 절차:
- 키보드를 이용해 모국어 문장을 입력하세요
- 선택한 언어로 번역된 결과를 받아보세요

기능:

- Apple 음성인식기술 지원 - 실시간 음성인식이 당신의 번역 속도를 크게 향상시켜줍니다
- 115가지 언어 문자 대 문자 번역
- 39가지 언어 음성 대 음성 번역
- 74가지 언어 음성 대 문자 번역
- 내장형 오프라인 음성: 인터넷 트래픽 사용을 줄이고 보다 빠르게 번역할 수 있습니다
- 직관적이고 스마트한 대화 및 번역 내역 저장 자주 사용하는 문구나 대화를 저장해둘 수 있는 어휘집으로도 사용할 수 있음
- iCloud 통합: 번역기록은 당신의 모든 Apple 장치를 아우르며 동기화됩니다
- 로그 테이프 편집 가능
- 클립보드를 이용해 로그 테이프 복사, 붙이기, 공유
- 이메일, SMS, Notes 앱 등으로 내역 로그 테이프 전체 내보내기 가능
- 설정 가능 옵션: 남성 및 여성 음성 선택, 읽기 속도, 대화 종료 자동 탐지, 번역 후 음성 출력 기능 등
