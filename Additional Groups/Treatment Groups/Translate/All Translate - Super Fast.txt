App Description
MAIN FEATURES

All translation was never so easy! Travel around the world and easily communicate in any country - Voice Translator will instantly pronounce translated phrases on-the-fly! Communicate freely at the airport, hotel, gas station, supermarket or any other place. Put your personal interpreter into pocket and use it anytime and anywhere!

All Translate is an essential learning tool for kids and adults - learn new phrases and listen to correct pronunciation.

Start using All Translate now and forget about looking for the correct phrase in the dictionary - say any phrase and your interpreter will pronounce the translation out loud!

Features:
- 100+ languages working in speech-to-speech and text-to-text modes
- Text translation: Translate between 110 languages by typing text.
- Speech translation: Best Text to speech and speech to text translation for any language.
- Audio translation: import or record audio to translate
- Image translation: import or camera shot get photo to translate
- ARKit translation: detect model use AR to translate
