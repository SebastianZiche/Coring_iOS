    New Features:

    (1) Fully supported on all iPod Touch. You can now use the built-in microphone on iPod Touch 4

    (2) Save 10-15% of battery when comparing to v1.0, and running faster at the same time

    (4) You can now send photos (hint: try sending photos, but be careful with what you send, to different agents and see what will happen)

    (8) Usage statistics added - Check network data consumed and message counts in the Settings tab

    (11) Allow password customization
