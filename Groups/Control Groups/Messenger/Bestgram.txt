App Description
Bestgram is an unofficial Telegram client.

By using Telegram’s API, Bestgram gives its users the best possible messaging experience with more than 500 million active users.

Bestgram also has some additional features for you. You can see these extra features below.

• Multi Forward: Edit and send messages to your friends at the same time.
• Confirmations: Confirm before some actions like sending a sticker or Gif
•Turbo: Automated proxy system for secure connection
