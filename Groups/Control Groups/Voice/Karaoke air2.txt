App Description
NOTE:Support ktvhost app version later than version20151008.

This application is used to control karaoke machine sold by our company,The KARAOKE machine is a Host device,user's iphone connect to karaoke machine by the same wireless router,after installing this application,input HOST IP address of machine or scan to connect,then user can control the karaoke machine on real-time basis, can order or delete a song.

functions:
1,can control play、pause、stop、replay、vocals/music switch、volume、tone and other specific controls.
2,you can access search function and input the name of a song or an artist with voice input function of iphone to start searching.
3,Multiple ways to order song: serving multinational language (UNICODE) and multiple ways to song order, differentiated by reference with alphabet、language、singer name、song words number、type of songs、song serial number、and so on. Each operation is easy and quick handle.
