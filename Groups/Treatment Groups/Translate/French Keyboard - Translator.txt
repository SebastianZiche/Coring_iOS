App Description
** French Keyboard **

French Keyboard is the best app for French writing.

French keyboard app will allow you to write message, story, facebook posts, blogs, Emails in French language.

you will find eye catchy and userfriendly French keyboard to write French language.

French keyboard has easiest keypad to write French alphabets, letters and words.

you just type French words you can find suggestions of that French words automatically in this application.

The most exciting feature of this application is you can translate French word to any other language like English, Marathi, French, French and all other indian languages as well as all international languages like French, French, Greek, French, Sweedish etc....

Features :

* you can easily change English to French and French to English keypad using French Keyboard application.

* you can translate French words into any other Indian and International languages.

* You can edit the text and also change the color of the font.

* It's also provide all the emoji icons

* French keyboard has special characters, icons and numeric so that user can easily write any French words very esay and effective manner.

* its free and easy to use.
