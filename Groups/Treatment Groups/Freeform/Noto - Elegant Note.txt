App Description
Features:
- Powerful text editing tools
- Bold / Italic / Underline / Strikethrough / Highlight
- Number list / Bullet list / Check list
- Unlimited attachments (Pro):
- Image / Video / Scan / Drawing / File
- Table / Block quote / Code / Math Equation
- Dark mode
- iCloud sync
- Organize with gesture
- Export to HTML, Markdown, PDF, & JPG
- Lock notes (Pro)
- Desktop class organization
- Drag to reorder your list and paragraph
- Drag to organize your notes
- Rapid multi select

Noto is free to use, including iCloud Sync support.

Unlock Noto Pro to enable all features including
* Attachments
* Note lock
* Export
* Support future Noto development

    • Noto will now restore to last opened folder and note when opened
    • Code block doesn't wrap lines
    • Updated document scanner to use system provided scanner
    • Math attachment now uses Katex as the rendering engine. Which supports more latex math instructions and new lines
    • New empty notes are automatically deleted
    • Add ability to toggle dark mode independently
    • Add mouse and trackpad support on ipadOS 13.4
    • Add horizontal rule
    • Add ability to import files and folders by dragging them into Noto's note browser. Supported types include:
    • .md
