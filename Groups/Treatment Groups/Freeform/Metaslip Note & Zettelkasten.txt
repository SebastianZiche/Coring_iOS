Productivity (Applications)

App Description
Metaslip is a minimalist note-taking software that focuses on the Zettelkasten, dedicated to helping users think deeply.

Features
* Privacy first, no network required, data stored locally.
* Data portability, can be exported as a universal Markdown file and SQLite database at any time.
* Robust data backup mechanism, supports manual export/import, WebDAV, and iCloud(Pro).
* Supports Markdown format, including LaTeX mathematical formulas.
* Supports bidirectional linking and parent-child linking.
* Supports card management by tags and projects.
* Deep optimization for Zettelkasten.
