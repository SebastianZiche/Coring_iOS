App Description
OkTalk Keyboard is a customizable keyboard that lets you translate everything you write into 60+ languages. The application is compatible with all popular messaging applications such as WhatsApp, iMessage, Telegram, Facebook Messenger, WeChat, Viber and others.

Features:
* Easy setup
* The keyboard seamlessly integrates with all applications
* Get easy translation in over 60 languages
* Works with your favorite instant messengers, social networks and email.
