App Description
This is a voice, text and image translator app that allows you to translate all languages like English, French to Vietnamese and many more.

Main features
1, Translate text
Real-time text translation for unlimited phrases and words
2, Translate images
Automatically recognize and translate text in images by taking pictures and pressing buttons, plus it can recognize many languages.
3, Voice translation
Real-time voice translation to both text and audio.
4, Offline translation
Can translate even in offline mode. Free network limit.
