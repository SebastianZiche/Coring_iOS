Translate Voice, Photo & Text

App Description
Translate Box can:

- Translate Photo
- Translate Voice
- Translate Text

With five Translators at once:
- Google Translate
- Microsoft Translator
- Yandex Translate
- Baidu Translate
- Naver Translate (Papago)

This app requires an internet connection.
