App Description
Powerful PDF tools at your fingertips. The latest PDF technology and software for a better experience. With an intuitive UI designed for delight – there’s a better way to PDF!

Do your best work with Drawboard.

- Dynamic PDF tools that feel good to use
- Multi-platform access and storage. Where you are, your work is.
- Paperless. A way of working for a sustainable future

TOOLS FOR EVERY JOB
Fast, responsive, and accurate markup tools. Draw with best-in-class ink for flawless Apple Pencil hand-feel and finally replace pen and paper.

CROSS-PLATFORM ACCESS ON ANY DEVICE
Upload your annotated PDFs to the cloud and mark up with others in real time with instant sync. Go Pro in-app for additional storage capacity.

WORK YOUR WAY
Customize your workspace by docking your favorite tools and build a toolbar that works for you!

MORE
- Text review markup - freehand highlighter, text highlighter, underline, strikeout, squiggly on PDF.
- Insert text and shapes - cloud tool, rectangle, ellipse, line, arrow, polygon and polyline on PDF.
- Insert images and camera photos onto PDFs.
- Create blank PDFs or from an image.
- Write, annotate, ink and draw on PDFs.
- Customize and edit annotation colors, stroke widths, and opacities.
- Create new blank PDF documents of different size, color, orientation and page count.
- Insert, rotate and delete PDF pages.
- Open, read, mark up and save large and complex PDF documents.

Version 2.0.0 (Feb 18, 2024)
    This version offers a completely overhauled document save system, to improve data reliability and prevent loss in the event of save failures, and even app crashes.

    * New autosave process in which changes are saved to a backup location within seconds.
    * New "Save to..." dialogs if saving to the original file fails, now you can save your changes to a new location.
    * Recovery dialogs when re-opening after a crash or save failure to restore unsaved changes.
    * Completely rewritten how we operate with third-party storage like iCloud, OneDrive and Google Drive including adding Revert and Reload options so you can pick up changes that may have occurred on another device.

    Plus many other technical improvements, including:

    * Faster updates on the canvas, particularly for pages with many annotations
    * Faster document loading including incremental loading for pages with thousands of annotations
    * Read-only documents can now be opened in-place instead of requiring a copy to a readable location
    * Fixed crashes that could occur when deleting annotations
    * Fixed crashes that could occur when uploading, renaming or moving files
    * Fixed deadlocks that could occur when other applications on the system attempted to read from a file that was open in Drawboard PDF
