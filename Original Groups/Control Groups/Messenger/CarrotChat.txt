App Description
CarrotChat is a highly secured encrepted IM software. Based on powerful end-to-end encryption algorithm, carrot chat can comprehensively, effectively and intelligently protect user communication data. As a secure IM software, CarrotChat has the following functions:

Security: All communication data of CarrotChat is encrypted into ciphertext through end-to-end encryption technology. It is kept in ciphertext during the whole transmission process to prevent communication data from being hecked. At the same time, CarrotChat also supports anti-screenshot, remote destruction, top secret sessions and other powerful information security functions.

High efficiency: Message receipt, group chat bulletin board, global search, special attention, only see someone, reply and other functions make CarrotChat more efficient in information exchange and processing.
