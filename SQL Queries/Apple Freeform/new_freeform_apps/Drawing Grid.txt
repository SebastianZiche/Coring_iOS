App Description
Drawing Grid app will help you keep proportion of drawing objects.
Simply set canvas size to correctly lay out individual grid elements and see physical dimensions, then set grid properties either as square blocks or blocks of specific size, exactly as you need them.
Finally, transform image by setting its position and changing its size.

If you are missing a feature just let us know by creating a new feature request from within the Drawing Grid app.
