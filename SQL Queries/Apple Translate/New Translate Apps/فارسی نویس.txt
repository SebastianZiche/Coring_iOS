App Promotional Text
The essential app for writing in Farsi or translating it into other languages!
App Description
Farsi Nevis keyboard extension simplifies Farsi typing effortlessly. It automatically transliterates Farsi words written in English letters (Pinglish/Finglish) to Farsi script as you type.

Key Features:

Streamlined design for easy use
Automatic transliteration
Universal keyboard compatible with all your apps
Full text translation from Farsi to various languages (Supported languages: English, French, Spanish, Turkish, Arabic, German, Dutch, Japanese, Korean, Russian, Swedish, Chinese)
Customize the keyboard with vibrant themes
Clipboard functionality
Number row for quick access
Emoji support
In-app converter for handling large text volumes
Upgrade now for a seamless Farsi typing experience!
